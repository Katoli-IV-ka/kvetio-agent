# ICP Summary — Справочник целевых сегментов

Краткое описание для агентов. Полная конфигурация — в `config/icp.yaml`.

## Что продаёт Kvetio

| Продукт | Покупатель |
|---|---|
| Разметка данных | Компании с ML-командами, которым не хватает размеченных данных |
| Готовые датасеты | Компании, которым нужны данные быстро — без сборки с нуля |

## Три основных сегмента (MVP)

### medical-imaging
Компании, обучающие модели на медицинских изображениях.
- **Что им нужно:** bounding boxes, сегментация, классификация патологий на рентгене / МРТ / гистологии
- **Ключевые слова:** DICOM, radiology AI, pathology AI, medical imaging, CT scan, mammography
- **Сигналы:** вакансии Clinical AI Engineer + ML Engineer, датасеты с медицинскими снимками на HuggingFace

### autonomous-vehicles
Компании, разрабатывающие восприятие для беспилотников, роботов, дронов.
- **Что им нужно:** lane detection, object tracking, 3D bounding boxes, семантическая сегментация видео/лидара
- **Ключевые слова:** autonomous driving, perception, lidar, SLAM, object detection, lane detection
- **Сигналы:** вакансии Perception Engineer / Computer Vision Engineer, репозитории с автономными системами

### speech-audio
Компании, разрабатывающие ASR, TTS, voice assistant, audio classification.
- **Что им нужно:** транскрипции, размеченные шумы, многоязычные аудиокорпуса
- **Ключевые слова:** speech recognition, ASR, TTS, voice cloning, audio classification, speaker diarization
- **Сигналы:** HuggingFace с аудио-моделями, вакансии Speech Engineer / Audio ML Engineer

## Признаки Hot-компании (≥60 баллов)

Одновременно присутствуют все четыре:
1. **Активная разработка AI-продукта** — свои модели, не просто API-обёртка
2. **Потребность в данных** — вакансии аннотаторов ИЛИ упоминание сборки датасетов
3. **Коммерческая состоятельность** — Seed+, есть ML-команда
4. **Свежесть** — активность в последние 30 дней

## Кто НЕ является целевым клиентом

- **Датасет-провайдеры** (Scale AI, Appen, Labelbox, Surge AI) — конкуренты
- **AI-обёртки** — компании, которые только вызывают OpenAI/Anthropic API
- **Большие корпорации** (Google, Meta, Apple) — у них свои команды разметки
- **Консалтинг / системные интеграторы** — перепродают чужие решения, не строят свои модели
- **Pre-product стартапы** без продукта и команды
